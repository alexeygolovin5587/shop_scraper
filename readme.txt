1. Description

	I scraped list of all categories from the three websites and analyze those information.

2. Run

	In shopscraper/, run:
	$ scrapy crawl souq
	$ scrapy crawl wadi
	$ scrapy crawl namshi

3. Result

	In shopscraper/data/, you can see 'souq_category.csv', 'wadi_category.csv' and 'namshi_category.csv' which have their category list.
